Trusted
Media Brands

Trusted Media Brands sought a modern, motion-forward experience for digital consumers to interact with their brand. It was developed using a headless Wordpress CMS and a React + PostCSS frontend.

Javascript, HTML, CSS, PostCSS, Tailwind, 11ty, Liquid, Wordpress

view live site

My Role

I worked with the design team to rapidly prototype the new Trusted Media Brands website. The initial prototype was built in a static environment using 11ty. When reviewed and approved internally, the static code was converted for use with the headless Wordpress CMS + React stack built by the back-end dev.

Content intended on each template page was expected to be varied, and so much care was put into building an implementation of the design that accounted for existence, or lack-thereof, of content on each page.

Interactions

Great care was placed in creating eye catching and performant interactions and animations. I worked alongside the design team to bring their ideas to life in a scope and performance friendly way.

Pure CSS

Nested CSS 3D transforms were used to create a responsive 3D grid. A dotted stroke is applied to an svg arch, and is animated using CSS keyframes. CSS variables are used to easily change the gradient to reflect the brand colors via a varible in the CMS.
